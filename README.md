# Portfolio — Attobra Prince Aman Junior

Portfolio personnel en HTML / CSS / JavaScript pur — sans framework.

## 🚀 Lancer le projet

Ouvrez simplement `index.html` dans votre navigateur, ou utilisez un serveur local :

```bash
# Python
python -m http.server 8000

# Node
npx serve .
```

Puis ouvrez http://localhost:8000

## 📸 Ajouter votre photo de profil

Placez votre photo dans `assets/profile.jpg` — elle apparaîtra automatiquement dans la carte hero.

1. Créer un repo sur GitHub
2. Push le contenu du dossier
3. Settings → Pages → Branch: main → Save
